# Master Index

Entry point for this knowledge vault's wiki. Each entry below links to a topic folder.

## Topics

_Empty — topic folders are created automatically when the agent first routes a captured note whose tags match the Tag → Topic mapping in the vault's `CLAUDE.md`._

## How this index works

- **New top-level topic** → a line gets added here with a one-line description of the topic.
- **Existing topic gets new content** → entry goes in the topic's own `wiki/[topic]/[topic].md`, not here.

Example of what this file looks like once populated:

```markdown
## Topics

- [[wiki/ai/ai|AI]] — AI agents, agentic workflows, LLM tools, prompt engineering
- [[wiki/development/development|Development]] — coding, automation, web dev, data science
- [[wiki/productivity/productivity|Productivity]] — PKM, workflow, knowledge management
```
